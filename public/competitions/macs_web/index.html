There is a database in the folder, try reading /var/www/html/mainsite/database_toBeFinished.db with stream wrappers, and the username and password (in MD5) are leaked:

joseph 0e087333482131113740957780965295

The code is vulnerable in its comparison between $databasePassword and md5("$password"). It uses loose type comparison and therefore it’s vulnerable to PHP Type Juggling. The output of the md5 function is a string and `$databasePassword`` value is going to be “0e087333482131113740957780965295” for user Joseph:

md5(string $string, bool $binary = false): string

The loose type comparison is pretty funky and you can read more about it here: PHP Loose Type. Basically:

0e08733 == 0e12345 is True So we have to find a password when hashed gives out a value that starts with 0e. Consult Magic Hash and we know we can use a value of 240610708 for the password. Login with username joseph/ password 240610708 and we can obtain the flag.
